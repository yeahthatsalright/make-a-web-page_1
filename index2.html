Concepts

Part One

Web Pages


Web pages are designed to give information using a code of HTML to make users able to see and access content by allowing browser to affectively communicate with servers


In Line Block Elements

In line lettering are things like "[br]" and "[span]" that are used as containers for text associated with HTML text with styling and other features.
Block elements examples would be things like "[p]" and [div] because they are used as a container for in-line elements to alter the visual nature of the web page.

Part Two

HTML and CSS

HTML stands for hypertext markup language and is styled by the CSS or cascading style sheet. The HTML controls what specific structure the web page will have while the CSS will control how the objects and text will look inside the structure

Document Object Model

The document object model or DOM refers to the tree like structure that a page is given through commands with the CSS styling. It refers to elements that can be altered by referring to its specific class name. A single class structure can alter the entire body of the HTML document or just a small portion.

Part Three

Boxifying

This term refers to the box like nature of HTML structure. Boxes can be repositioned within the structure by finding its corresponding "div" and applying a “display: flex” within the CSS document to override the linear design that it accustomed to. Boxes can be controlled through the CSS by changing the margining or how the border will interact with the text.

Verifying Code

Several free online code verification services are available to those that need it. Since programmers are likely to make simple mistakes or repeat unnecessary value, it is always handy to have an online resource the can debug and tell you where you might have made errors.

Avoiding Repetition

When writing code, it is always a great idea to avoid repetition when writing your code. This basically means using various class structure and command runs to simplfy your code. In javascript this is done with setting up things like data arrays.
